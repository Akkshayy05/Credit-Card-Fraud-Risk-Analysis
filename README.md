# Credit-Card-Fraud-Risk-Analysis

## Tools Used
- PostgreSQL
- SQL (CTE, Window Functions, NTILE)
- Power BI (DAX, KPI Dashboard)

## Dataset
284,806 financial transactions

## Key Findings
- Overall fraud rate: 0.17%
- Fraud revenue impact: 0.24%
- High-value transactions had 2x higher fraud probability (0.38%)
- Top 1% transactions showed elevated fraud risk

## Skills Demonstrated
- Risk segmentation
- Window functions
- Percentile analysis
- Dashboard storytelling
